# Exemplar Gallery (navigational index)

**What this is.** A map of the **annotated corpus exemplars** that live inside the
section pattern files. It is an *index, not a relocation* — the annotated verbatim
exemplars stay in their pattern files because they are the verifiability chain for every
distilled rule (`corpus_manifest.md` §2). This page only tells you *which exemplar to
study for which paper, and where it is*.
