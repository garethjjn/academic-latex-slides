# Journal Profile Bank (single source)

**What this is.** The canonical home for JAE / JAR / TAR / AJPT conventions, previously
scattered across the hub's "Default journal targeting" table, `intro_patterns.md`'s
roadmap rule, `abstract`'s length rules, and `audit-referee-response`'s reviewer-culture
section. Sub-skills keep their one-line application and link here for the full profile.

Provenance / "prior not law" caveat: `corpus_manifest.md` §2. These profiles describe
the named corpus + reviewer-culture distilled from it; treat as strong defaults a writer
may override with reason, not as journal policy.

---

## Per-journal profile

| Dimension | **JAE** | **JAR** | **TAR** | **AJPT** |
|---|---|---|---|---|
| Roadmap (end of intro) | Drop (modern, 2020+) | Optional — check 2–3 recent area papers | Keep, short 3-sentence | Keep |
| Abstract magnitudes | None (dir. only) | None | None | None |
| Abstract length | ~100–180 w | ~100–180 w | ~100–180 w | ≤200+ tolerated |
| Identification bar | Highest — probes causal hard | High — econometric care, SE/FE/sample | Medium — theory-empirics linkage | Lower — practitioner relevance |
| Contribution-consciousness | High | **Highest** ("what's NEW vs prior?") | Medium (institutional richness valued) | Lower (replications OK) |
| Methodology innovation | Welcomed (with construct validity) | Less; cleanliness > novelty | Welcomed if theory-linked | Refinements OK |
| Over-claim tolerance | Low | Lowest (flags "show that/prove") | Low | Low |
| Descriptive/correlational | Needs identification | Needs identification | Tolerated if institutional contribution strong | Tolerated |

## Reviewer culture (what each desk probes first)

- **JAE** — identification rigor (shocks, rotation, falsification, FE saturation);
  magnitude-grounded results; connection to the audit-quality framework.
- **JAR** — design cleanliness (SE clustering, FE structure, sample selection); sharp
  contribution articulation vs the closest prior paper; wide robustness battery.
- **TAR** — institutional richness; explicit theory→specification mapping; clear
  organization and signposting (roadmaps welcomed).
- **AJPT** — practitioner relevance ("what should auditors do differently?"); less
  novelty-obsessed.

## When the journal is unspecified — inference cues

| Cue from the input | Default assumption |
|---|---|
| US data, partner-trait, post-2017 | JAR |
| Cross-country / institutional comparison | JAE |
| Single-country regulatory experiment | JAE (occasionally TAR) |
| Theory/framework-led paper | TAR |
| Audit-fee / practitioner focus | AJPT (tone slightly less restrictive) |
| Short-paper / replication | JAR |

If still unsure, **ask**. The roadmap rule and several structural defaults hinge on this.
