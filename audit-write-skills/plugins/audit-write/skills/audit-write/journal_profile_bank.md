# Journal Profile Bank (single source)

**What this is.** The canonical home for the **accounting top-5** conventions —
JAE / JAR / TAR / CAR / RAST — previously scattered across the hub's "Default journal
targeting" table, `intro_patterns.md`'s roadmap rule, `abstract`'s length rules, and
`audit-referee-response`'s reviewer-culture section. Sub-skills keep their one-line
application and link here for the full profile.

> **Scope note.** The suite targets the five top-tier *accounting* journals
> (JAE, JAR, TAR, CAR, RAST). It no longer profiles AJPT (*Auditing: A Journal of
> Practice and Theory*) as a target desk — AJPT is a strong field journal but outside the
> top-5 archival set this suite calibrates to. (AJPT still appears as a real-journal
> *citation* where a source paper was published there, e.g. Nelson & Tan 2005.)

Provenance / "prior not law" caveat: `corpus_manifest.md` §2. These profiles describe
the named corpus + reviewer-culture distilled from it; treat as strong defaults a writer
may override with reason, not as journal policy.

## Per-journal profile

| Dimension | **JAE** | **JAR** | **TAR** | **CAR** | **RAST** |
|---|---|---|---|---|---|
| Roadmap (end of intro) | Drop (modern, 2020+) | Optional — check 2–3 recent area papers | Keep, short 3-sentence | Drop (modern) | Drop (modern) |
| Abstract magnitudes | None (dir. only) | None | None | None | None |
| Abstract length | ~100–180 w | ~100–180 w | ~100–180 w | ~100–180 w | ~100–150 w (papers run shorter) |
| Identification bar | Highest — probes causal hard | High — econometric care, SE/FE/sample | Medium — theory-empirics linkage | High — clean design, near JAR | High — capital-markets rigor |
| Contribution-consciousness | High | **Highest** ("what's NEW vs prior?") | Medium (institutional richness valued) | High (clear *incremental* contribution) | High (economic-significance framed) |
| Methodology innovation | Welcomed (with construct validity) | Less; cleanliness > novelty | Welcomed if theory-linked | Welcomed (methodologically pluralistic) | Welcomed if economically/theory framed |
| Over-claim tolerance | Low | Lowest (flags "show that/prove") | Low | Low | Low |
| Descriptive/correlational | Needs identification | Needs identification | Tolerated if institutional contribution strong | Needs identification (some tolerance if well-motivated) | Needs identification / economic framing |

## Reviewer culture (what each desk probes first)

- **JAE** — identification rigor (shocks, rotation, falsification, FE saturation);
  magnitude-grounded results; connection to the audit-quality framework.
- **JAR** — design cleanliness (SE clustering, FE structure, sample selection); sharp
  contribution articulation vs the closest prior paper; wide robustness battery.
- **TAR** — institutional richness; explicit theory→specification mapping; clear
  organization and signposting (roadmaps welcomed).
- **CAR** — rigorous, clean archival design (SE clustering, FE structure, sample
  selection) paired with a clearly articulated *incremental* contribution;
  methodologically pluralistic (archival, analytical, experimental all at home);
  values broad/international generalizability.
- **RAST** — the financial-reporting-quality / capital-markets linkage; economic
  significance and a clear economic or theoretical frame for the prediction; concise
  execution (papers run shorter); analytical-empirical work valued.

## When the journal is unspecified — inference cues

| Cue from the input | Default assumption |
|---|---|
| US data, partner-trait, post-2017 | JAR |
| Cross-country / institutional comparison | JAE |
| Single-country regulatory experiment | JAE (occasionally TAR) |
| Theory/framework-led paper | TAR |
| Clean archival audit-quality paper, broad method, incremental (not field-defining) contribution | CAR |
| Audit ↔ financial-reporting-quality / capital-markets linkage; analytical-empirical; concise | RAST |
| Short-paper / replication | JAR |

If still unsure, **ask**. The roadmap rule and several structural defaults hinge on this.
