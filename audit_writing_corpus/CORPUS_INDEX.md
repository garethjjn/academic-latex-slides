## Skill targets these papers feed

| Skill | What we extract |
|---|---|
| `audit-write` (master) | Audit quality conceptual framework, hedging vocabulary, journal-culture cues |
| `audit-write-abstract` | DeFond / Zuo abstract micro-structure (5 moves in ~150 words) |
| `audit-write-intro` | 5-paragraph intro template (motivation → gap → setting → finding → contribution) |
| `audit-write-hypothesis` | Audit-quality causal-chain rhetoric, alternative hypotheses |
| `audit-write-design` | Identification rhetoric, audit-specific endogeneity language |
| `audit-write-results` | Effect-size language, null-result handling, economic-magnitude framing |
| `audit-write-robustness` | DeFond standard battery: subsamples, PSM (post-2017), entropy balance, placebo |
| `audit-referee-response` | JAR / JAE / TAR reviewer-culture language |
